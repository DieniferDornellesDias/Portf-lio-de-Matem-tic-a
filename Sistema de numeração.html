Sistemas de Numeração

Sistema de numeração é um conjunto de símbolos que usamos para representar as quantidades numéricas. A base é a quantidade de número/dígitos que serve para representar o sistema, serve para representar o símbolo do sistema.A seguir vou falar um pouco sobre cada sistema de numeração alguns deles são:
  • Sistema Decimal
  • Sistema Binário
  • Sistema Octal
  • Sistema Hexadecimal

Sistema Decimal

O sistema decimal é de base dez, ou seja é usado dez símbolos (algarismos) diferentes para representar todos os números é um sistema posicional, a posição do número altera o seu valor.

Exemplo de uma decomposição de um número decimal

    Algumas regrinhas que podem ajudar

    0 + 0 = 0
    1 + 0 = 1
    1 + 1 = 10 → fica o “0” e o “1” é transportado para a próxima casa
    1 + 1 + 1 = 11 → fica o “1” e o outro “1” é transportad

    0 – 0 = 0
    1 – 0 = 1
    1 – 1 = 0
    0 – 1 = 1 Nesse caso, devemos “pedir emprestado”, daí fica 10 – 1 = 1

    0 ∙ 0 = 0
    1 ∙ 0 = 0
    1 ∙ 1 = 1

    Para realizar as conversões é simples,vamos ver como transformar um número que está escrito na Base 10 para a Base 2.

    Regras das divisões sucessivas por 2
    1º Fazer a divisão do número decimal por 2 e guardar o resto.

    Como realizar algumas conversões?


    Conversão da base 10 para a base 16, é só fazer o mesmo procedimento na conversão de Base 10 para Base 2, só que fazendo as divisões sucessivas por 16.


    Regra das divisões sucessivas por 16
    1º Fazer a divisão do número decimal por 16 e guardar o resto.
    2º Dividir o quociente novamente por 16 e guardar o resto.
    3º Repetir o processo até termos o quociente menor do que 16
    4º O número na base 16 será a sequência de restos obtidos juntamente com o último quociente, começando pela direita.
    Mas quando tivermos os restos 10, 11, 12, 13, 14, e 15, substituir pelos símbolos A, B, C, D, E e F, respectivamente.

  • ExemploConverta os números na base 10 para a base 16

    • Conversão: Base 16 para Base10


    O procedimento é o mesmo realizado na conversão de Base 2 e 8 para a Base 10 É só decompor o número em potências de 16 conforme a posição de seus dígitos.

    Regra da decomposição de número na Base 16
    Cada dígito hexadecimal deve ser multiplicado por potências de 16. O último dígito é multiplicado por 160 , o penúltimo por 161 , o antepenúltimo por 162 e, assim, sucessivamente. Quando o número tiver as letras A, B, C, D, E e F, substituir por 10, 11, 12, 13, 14 e 15, respectivamente.
